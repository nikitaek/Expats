## Процесс 2: Поиск и анализ гостиниц через туроператоров

**Описание:**
Система автоматически собирает информацию о гостиницах, используемых русскоязычными туроператорами для отправки туристов во Вьетнам, и формирует единую базу данных с географической и коммерческой аналитикой.

### Этап 1. Поиск туроператоров

Система регулярно определяет основных туроператоров, работающих с туристическим потоком из России, Беларуси, Казахстана, Узбекистана и других русскоязычных стран во Вьетнам.

Для каждого оператора сохраняются:

* Название оператора
* Сайт оператора
* Страна оператора

### Этап 2. Сканирование туров

Система сканирует сайты туроператоров и собирает все доступные туры во Вьетнам.

Из каждого тура извлекаются:

* Город назначения
* Гостиница
* Категория гостиницы
* Дополнительные доступные данные

### Этап 3. Построение базы гостиниц

После агрегации данных формируется единый список гостиниц.

Для каждой гостиницы производится объединение данных из всех найденных источников.

### Этап 4. Обогащение данных

Для каждой гостиницы система пытается определить:

* Точный адрес
* Город
* Количество звезд
* Географические координаты
* Приблизительный ценовой сегмент

Для поиска координат используются геокодинг-сервисы и внешние источники данных.

### Этап 5. Анализ популярности гостиниц

Для каждой гостиницы рассчитываются:

* Количество туроператоров, продающих данную гостиницу
* Список туроператоров, продающих данную гостиницу
* Частота упоминания гостиницы среди всех найденных туров

Это позволяет определить наиболее популярные гостиницы среди русскоязычных туристов.

### Этап 6. Сегментация гостиниц

Каждой гостинице присваивается условный уровень от 1 до 10 на основании предполагаемой стоимости проживания.

Назначение сегмента:

* 1–3 — бюджетный сегмент
* 4–6 — средний сегмент
* 7–8 — высокий сегмент
* 9–10 — премиальный сегмент

Сегментация используется для оценки платежеспособности туристов, проживающих в гостинице.

### Итоговый набор данных

Для каждой гостиницы сохраняются следующие поля:

* Hotel Name
* Address
* City
* Latitude
* Longitude
* Stars
* Price Segment (1–10)
* Operators Count
* Operators List

### Этап 7. Визуализация

Финальная база экспортируется в CSV-файл.

CSV автоматически импортируется в Google My Maps для визуализации гостиниц на карте и дальнейшего анализа туристической инфраструктуры.
